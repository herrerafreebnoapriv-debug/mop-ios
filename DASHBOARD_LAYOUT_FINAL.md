# Dashboard 布局最终规格

## ✅ 确认的布局结构

**左侧系统菜单 + 右侧4列主内容区域**

```
┌──────────────────────────────────────────────────────────────┐
│  Header: MOP 主控制台 | 系统管理员 [已连接] | 刷新 | 退出      │
├──────────┬────────────────────────────────────────────────────┤
│          │  Grid: 4列布局                                      │
│ 系统菜单 │  ┌──────┬──────┬──────────────┬──────┐            │
│ (固定)   │  │ 列1  │ 列2  │   列3-4      │      │ 行1        │
│          │  │用户  │创建  │  我的房间     │      │            │
│          │  │信息  │房间  │  (跨2列)      │      │            │
│          │  ├──────┼──────┼──────────────┼──────┤            │
│          │  │ 列1  │ 列2  │   列3-4      │ 列4  │ 行2        │
│          │  │设备  │系统  │  我的房间     │房间  │            │
│          │  │管理  │统计  │  (继续)       │二维码│            │
│          │  └──────┴──────┴──────────────┴──────┘            │
│          │  ┌──────────────────────────────────────────────┐ │
│          │  │  用户管理（跨4列，仅管理员可见）                │ │
│          │  └──────────────────────────────────────────────┘ │
│          │  ┌──────────────────────────────────────────────┐ │
│          │  │  邀请码管理（跨4列，仅超级管理员可见）          │ │
│          │  └──────────────────────────────────────────────┘ │
└──────────┴────────────────────────────────────────────────────┘
```

## 📐 4列布局详细分配

### 第一行
- **列1**：用户信息 (`grid-column: 1; grid-row: 1;`)
- **列2**：创建房间 (`grid-column: 2; grid-row: 1;`)
- **列3-4**：我的房间 (`grid-column: 3 / 5; grid-row: 1 / 3;`) - 跨2列2行

### 第二行
- **列1**：设备管理 (`grid-column: 1; grid-row: 2;`)
- **列2**：系统统计 (`grid-column: 2; grid-row: 2;`)
- **列3-4**：我的房间（继续，已跨行）
- **列4**：房间二维码 (`grid-column: 4; grid-row: 2;`)

### 第三行（全宽）
- **用户管理**：`grid-column: 1 / 5; grid-row: 3;` - 跨4列

### 第四行（全宽）
- **邀请码管理**：`grid-column: 1 / 5; grid-row: 4;` - 跨4列

## 🎨 CSS Grid 实现

```css
.main-content {
    display: grid;
    grid-template-columns: repeat(4, 1fr); /* 4列等宽 */
    gap: 20px;
    padding: 20px;
}

/* 卡片样式 */
.card {
    background: white;
    border-radius: 8px;
    padding: 20px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

/* 用户信息 */
#user-info-card {
    grid-column: 1;
    grid-row: 1;
}

/* 创建房间 */
#create-room-card {
    grid-column: 2;
    grid-row: 1;
}

/* 我的房间（跨2列2行） */
#rooms-list-card {
    grid-column: 3 / 5;
    grid-row: 1 / 3;
}

/* 设备管理 */
#devices-card {
    grid-column: 1;
    grid-row: 2;
}

/* 系统统计 */
#stats-card {
    grid-column: 2;
    grid-row: 2;
}

/* 房间二维码 */
#qrcode-card {
    grid-column: 4;
    grid-row: 2;
}

/* 用户管理（跨4列） */
#user-management-card {
    grid-column: 1 / 5;
    grid-row: 3;
}

/* 邀请码管理（跨4列） */
#invitation-management-card {
    grid-column: 1 / 5;
    grid-row: 4;
}
```

## 📱 响应式设计

### 桌面端（> 1200px）
- 4列布局
- 左侧菜单：200-250px 固定宽度

### 平板端（768px - 1200px）
- 3列布局（合并第3-4列）
- 左侧菜单：可折叠

### 移动端（< 768px）
- 单列布局
- 左侧菜单：隐藏（汉堡菜单）

## ✅ 确认事项

- ✅ 左侧系统菜单栏（固定宽度）
- ✅ 右侧主内容区域：4列布局
- ✅ 我的房间卡片：跨2列2行
- ✅ 用户管理和邀请码管理：跨4列全宽
- ✅ 所有功能根据权限显示/隐藏
