# Dashboard 4列布局详细规格

## 📐 布局结构

### 整体结构
```
┌──────────────────────────────────────────────────────────────┐
│  Header (全宽)                                               │
├──────────┬───────────────────────────────────────────────────┤
│          │  Grid: 4列布局                                     │
│ 系统菜单 │  ┌──────┬──────┬──────────────┬──────┐           │
│ (固定)   │  │ 列1  │ 列2  │   列3-4      │ 列4  │ 行1       │
│          │  │用户  │创建  │  我的房间     │房间  │           │
│          │  │信息  │房间  │  (跨2列)      │二维码│           │
│          │  ├──────┼──────┼──────────────┼──────┤           │
│          │  │ 列1  │ 列2  │   列3        │ 列4  │ 行2       │
│          │  │设备  │系统  │  我的房间     │      │           │
│          │  │管理  │统计  │  (继续)       │      │           │
│          │  └──────┴──────┴──────────────┴──────┘           │
│          │  ┌──────────────────────────────────────────────┐ │
│          │  │  用户管理（跨4列）                            │ │
│          │  └──────────────────────────────────────────────┘ │
│          │  ┌──────────────────────────────────────────────┐ │
│          │  │  邀请码管理（跨4列）                            │ │
│          │  └──────────────────────────────────────────────┘ │
└──────────┴───────────────────────────────────────────────────┘
```

## 🎯 CSS Grid 规格

### Grid 定义
```css
.main-content {
    display: grid;
    grid-template-columns: repeat(4, 1fr); /* 4列等宽 */
    gap: 20px; /* 卡片间距 */
    padding: 20px;
}
```

### 卡片位置分配（4列布局）

#### 方案一：房间二维码在第一行右侧
```
第一行（4列）：
- 列1：用户信息 (grid-column: 1; grid-row: 1;)
- 列2：创建房间 (grid-column: 2; grid-row: 1;)
- 列3-4：我的房间 (grid-column: 3 / 5; grid-row: 1 / 3;) 跨2列2行
- 列4：房间二维码 (grid-column: 4; grid-row: 1;) 
  ⚠️ 注意：与我的房间重叠，需要调整

第二行（4列）：
- 列1：设备管理 (grid-column: 1; grid-row: 2;)
- 列2：系统统计 (grid-column: 2; grid-row: 2;)
- 列3-4：我的房间（继续，已跨行）
- 列4：（空）
```

#### 方案二：房间二维码在第二行右侧（推荐）
```
第一行（4列）：
- 列1：用户信息 (grid-column: 1; grid-row: 1;)
- 列2：创建房间 (grid-column: 2; grid-row: 1;)
- 列3-4：我的房间 (grid-column: 3 / 5; grid-row: 1 / 3;) 跨2列2行

第二行（4列）：
- 列1：设备管理 (grid-column: 1; grid-row: 2;)
- 列2：系统统计 (grid-column: 2; grid-row: 2;)
- 列3-4：我的房间（继续，已跨行）
- 列4：房间二维码 (grid-column: 4; grid-row: 2;)
```

**推荐使用方案二**，避免卡片重叠。

#### 第三行（全宽）
- **用户管理**：`grid-column: 1 / 5; grid-row: 3;` (跨4列)

#### 第四行（全宽）
- **邀请码管理**：`grid-column: 1 / 5; grid-row: 4;` (跨4列)

## 📱 响应式断点

### 桌面端（> 1200px）
- 4列布局
- 左侧菜单栏：200-250px
- 卡片间距：20px

### 平板端（768px - 1200px）
- 3列布局（合并第3-4列为1列）
- 左侧菜单栏：可折叠，150px
- 卡片间距：15px

### 移动端（< 768px）
- 单列布局
- 左侧菜单栏：隐藏（汉堡菜单）
- 卡片间距：10px

## 🎨 卡片尺寸建议

- **标准卡片**：1列宽
- **我的房间**：2列宽，2行高（根据内容自适应）
- **全宽卡片**：4列宽（用户管理、邀请码管理）

## ⚠️ 注意事项

1. **我的房间卡片**：根据图片描述，这个卡片是垂直跨越的，可能需要特殊处理
2. **房间二维码**：需要确认是放在第一行右侧还是第二行右侧
3. **响应式**：确保4列布局在小屏幕上能正确折叠
4. **左侧菜单**：固定定位，内容滚动时保持可见
